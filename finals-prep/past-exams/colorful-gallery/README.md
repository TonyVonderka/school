# Praktická maturitní zkouška - Barevná galerie

**Autor:** Antonín Vonderka  
**Datum:** 2026 / 04

## Web

Tato část dokumentuje tvorbu responzivní webové stránky, která slouží jako interaktivní galerie obrázků. Byla vytvořena pomocí technologií HTML, CSS a JavaScript. Web nabízí uživatelům možnost přepínat mezi třemi různými barevnými motivy a prohlížet si obrázky v plném rozlišení prostřednictvím modálního okna.

- [🖥️ Přejít na dokumentaci Webu](./documentation/web.md)

## Databáze

Tato sekce obsahuje konceptuální návrh (E-R diagram), strukturu entitních typů a inicializační SQL skripty pro relační databázi. Databáze je navržena pro evidenci uživatelů, nahrávání obrázků a správu komentářů.

- [💾 Přejít na dokumentaci Databáze](./documentation/db.md)

## Program

Tato část přibližuje konzolovou aplikaci napsanou v jazyce Python. Program slouží jako kalkulačka pro vyhodnocování matematických výrazů zapsaných v reverzní polské notaci (RPN). Obsahuje také ošetření chybových stavů, jako je dělení nulou nebo neplatný formát vstupu.

- [⚙️ Přejít na dokumentaci Programu](./documentation/prog.md)
