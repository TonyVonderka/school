# Beam of Fear

**Autor:** Antonín Vonderka

Atmosferická hororová hra z pohledu první osoby vytvořená v Unity.

## 🚀 Jak hru spusit

1. Stáhněte si nejnovější verzi v sekci [Releases](https://github.com/TonyVonderka/school/releases)
2. Rozbalte zip a spusťte `BeamOfFear.exe`
